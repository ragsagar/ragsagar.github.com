Script to reboot teracom router

Sat 13 August 2011
By Rag Sagar.V

A slightly edited version of the script to restart router written by Madhusudan.C.S to make it work with teracom router given by bsnl. [sourcecode language="bash"] #!/usr/bin/env expect set username admin set pass admin set host 192.168.1.1 spawn telnet ${host} expect -re "Login:" send "${username}\r" expect "Password:" send "${pass}\r" expect -re "successful" send "console enable\r" send "restart\r" expect eof [/sourcecode]
